### Step-by-Step Guide for Installing Minecraft Forge Mods

Step 1: Download the Minecraft Forge Installer

The first step in installing Minecraft Forge mods is to download the installer from the official Forge website. Make sure you download the version that corresponds with your Minecraft version. For example, if you’re playing Minecraft 1.18, download the Minecraft Forge 1.18 installer.

Step 2: Run the Minecraft Forge Installer

Once you’ve downloaded the installer, run it and follow the prompts to install Minecraft Forge. Remember to select the correct Minecraft version during the installation process.

Step 3: Launch Minecraft Using the Forge Profile

After installation, launch Minecraft and select the Forge profile. This will launch Minecraft with Forge installed.

Step 4: Add Mods to the Mods Folder

To add mods, download the mod you want and place it in the mods folder in your Minecraft directory. The mods folder can be found in the .minecraft folder, which is typically located in your AppData directory.

Step 5: Launch Minecraft Again

After you’ve added your mods, launch Minecraft again using the Forge profile. Your mods should be loaded and ready to use!
