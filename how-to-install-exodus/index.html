How to Install Exodus on Kodi

The first thing you need to do is make sure that you have Kodi installed on your device. Then, follow these steps:

Step 1: Enable Unknown Sources

Before you can install any third-party add-ons, you need to enable installations from unknown sources. To do this, open Kodi and go to Settings > System Settings > Add-ons. Then, toggle on the option that says “Unknown Sources”.

Step 2: Add the Kodi Bae Repository

The next step is to add the Kodi Bae Repository, which will allow you to install Exodus. To do this, go back to the previous screen and click on “File Manager”. From there, select “Add Source”, and then enter the following URL: http://lazykodi.com. Name the source “Kodi Bae” and hit “OK”.

Step 3: Install the Kodi Bae Repository

Now that you’ve added the Kodi Bae Repository as a source, you need to install it. Return to the home screen of Kodi and select “Add-ons”. Then, click on the open box icon in the top left corner, which will take you to the “Add-on Browser” screen. From here, select “Install from zip file”, and then choose the “Kodi Bae” source that you just added. Finally, select the file named “repository.kodibae-x.x.zip”.

Step 4: Install Exodus

Now that you have the Kodi Bae Repository installed, you can use it to install Exodus. From the “Add-on Browser” screen, select “Install from repository”. Then, choose “Kodi Bae Repository” > “Video add-ons” > “Exodus” > “Install”.

Step 5: Enjoy Your Streaming

Once the installation is complete, you’ll find Exodus in the “Add-ons” section of Kodi. Simply click on it to start streaming your favorite TV shows and movies.
