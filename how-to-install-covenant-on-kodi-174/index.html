How to Install Covenant on Kodi 17.4

The first step is to enable unknown sources in Kodi. This allows you to install addons from third-party repositories.

To enable unknown sources, go to System Settings > Add-ons > Unknown sources.

Next, you need to add the repository for Covenant. You can do this by following these steps:

  1. Go to File Manager > Add Source.
  2. Enter the following URL: http://archive.org/download/repository.xvbmc/ and name it something like “XvBMC”.
  3. Go to Add-ons and click the package installer icon.
  4. Click “Install from zip file” and select the source you just added (“XvBMC” in this case).
  5. Click on “repository.xvbmc-4.2.0.zip” to install the repository.

Once the repository is installed, you can now install Covenant. Follow these steps:

  1. Go to Add-ons > Install from repository.
  2. Select “XvBMC (Add-ons)” repository.
  3. Select “Video add-ons.”
  4. Find “Covenant” and click “Install.”
  5. Wait for the installation to finish, and then start using Covenant.
