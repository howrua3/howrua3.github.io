How to Install Kodi on Amazon Fire

To install Kodi on Amazon Fire, follow these simple steps:

Step 1: Turn on your Amazon Fire TV

Make sure your Amazon Fire is turned on and connected to the internet.

Step 2: Enable Unknown Sources

Go to Settings > Device > Developer Options > Apps from unknown sources, and switch it on.

Step 3: Download the Downloader App

Return to the home screen and press the search button. Type in “Downloader” and click on the first option that comes up. Download and install the app.

Step 4: Install Kodi on Fire TV Stick
