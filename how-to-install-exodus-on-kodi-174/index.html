The installation process for Exodus on Kodi 17.4 is straightforward. Here are the steps:

  2. Select ‘Settings’
  3. Select ‘File Manager’
  4. Select ‘Add Source’
  5. Type in ‘http://lazykodi.com/'
  6. Name the source ‘Lazy Kodi’
  7. Go back to the home screen and select ‘Add-ons’
  8. Select the open box icon on the top left to access the Add-on browser
