How to Install SuperRepo on Kodi

Before we get started, make sure you have Kodi installed on your device. Now let’s dive into the installation process:

  1. Launch Kodi and click on the gear icon to access the settings menu.
  2. Click on the “File Manager” option and then click “Add Source”.
  3. Enter the following URL: https://superrepo.org/kodi/ and name the source “SuperRepo”.
  4. Return to the main menu and navigate to “Add-ons”.
  5. Click on the “Package Installer” icon in the top-left corner and select “Install from Zip File”.
  6. Select the “SuperRepo” source and click on “Krypton” or “Jarvis”, depending on your version of Kodi.
  7. Select “All”, then select “superrepo.kodi.krypton.all-x.x.x.zip”.
  8. Wait for the add-on to install and then navigate to “Add-ons” again.
  9. Click on “Install from Repository” and select “SuperRepo All”.
  10. Select the category of add-ons you’d like to install, such as “Video Add-ons” or “Music Add-ons”.
  11. Select the specific add-ons you want to install and click “Install”.
  12. Wait for the installation to complete and enjoy your new add-ons!
