The Solution: Installing Kodi on Firestick 2017

Installing Kodi on firestick 2017 isn’t as complicated as you might think. To begin, you’ll need to enable “Apps from unknown sources” on your Firestick before you can download Kodi. Then you can download and install Kodi on your Firestick via either ES File Explorer or the Downloader app. ES File Explorer is a file manager app that lets you manage your files efficiently, while the Downloader app lets you download files on your Firestick. Both apps are available on the Amazon App Store, and you can start the installation process by searching for them using the Alexa voice remote.

Step-by-Step Guide on Installing Kodi on Firestick 2017

So, let’s dive deep into the step-by-step guide to install Kodi on a Firestick 2017 device. First, we need to ensure that “Apps from unknown sources” is enabled. To do this:

1. Go to “Settings” on your Firestick home screen, then select “Device.”

2. From “Device,” click on “Developer Options.”

3. Scroll down and enable “Apps from unknown sources.”

4. A warning message will appear - click “OK.”

Once you have enabled “Apps from unknown sources.” Now we need to download and install either ES File Explorer or the Downloader app.ES File Explorer is a popular file manager app that is available on the Amazon App Store. Just follow these steps to download and install it:

1. From the Home screen, navigate to the search bar and type in “ES File Explorer.”

2. Select the ES File Explorer app from the search results.

3. Click on the “Download” button and wait for the installation process to complete.

After downloading and installing ES File Explorer, you can now proceed with downloading and installing Kodi on Firestick 2017:

1. Launch ES File Explorer on your Firestick.

2. Click on the three-lined menu button, then click on “Tools.”

3. Select “Download Manager” and click on “New” to add a source.

4. A new window will appear - type in the following URL exactly - “http://kodi.tv/download."

5. Give the source a name “Kodi,” click on “Download Now,” wait for the download to complete, and finally click on “Open File.”

6. Follow the prompt, and the installation process will begin automatically. Wait for the installation to complete, and that’s it! You have successfully installed Kodi on Firestick 2017.

How to Update Kodi on Firestick 2017

Since Kodi is open-source software, you need to update it regularly to ensure its smooth operation. The latest version of Kodi is Kodi 19, and you can download the latest version by following these steps:

1. Launch Kodi on your Firestick.

2. Click on the “Settings” icon.

3. Click on “System,” then “System Info.”

4. Click on “Check for updates.”

5. If a new version of Kodi is available, click on “Install” and wait for the update to complete.
