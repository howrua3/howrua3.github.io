How to Install Fusion on Kodi

Let’s start with the basic steps and pre-requisites to install Fusion on Kodi. Before you proceed, make sure Kodi is installed on your device, and you have an active internet connection. Once you have this, simply follow these steps:

  1. Launch Kodi on your device and click on the gear icon in the top left corner to open the settings menu.
  2. Select the “System” option and find the “Add-ons” tab on the left-hand menu.
  3. Enable the option “Unknown sources” from the right pane. This option allows you to install the Fusion addon, and it will warn you about potential risks.
  4. Click “Yes” to confirm the warning message.
  5. Return to the Kodi home screen and select the “Settings” icon again. This time, scroll down to “File manager” and select it.
  6. Click the “Add source” option on the left panel.
  7. In the pop-up window, click on “None” and enter the following URL: https://fusion.tvaddons.co
  8. Enter “Fusion” as the name for this media source and click OK to save your changes.
  9. Go back to the Kodi home screen and select “Add-ons” from the left pane.
  10. Click the “box” icon at the top of the screen to open the addon browser.
  11. Select “Install from zip file” and locate the “Fusion” source you just added.
  12. Select the “begin-here” folder and choose the file “plugin.program.addoninstaller-1.2.9.zip”.
  13. Wait for the installation confirmation message to appear on the screen.
  14. Return to the Kodi home screen and select “Programs” from the left pane.
  15. Choose the “Addon Installer” option and click on “Featured Add-ons.”
  16. Look for the “Fusion Repository” and click on it to open its contents.
