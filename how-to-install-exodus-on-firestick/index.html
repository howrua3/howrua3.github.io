How to Install Exodus on Firestick

The first step in installing Exodus on Firestick is to enable installation from unknown sources in the Firestick settings menu. Go to Settings>My Fire TV>Developer Options and then turn on the option for “Apps from Unknown Sources.” Next, you will need to download and install the ES File Explorer app from the Amazon App Store. After ES File Explorer is installed, you can use it to download and install the Kodi media player. Once Kodi is installed, you can then add the Exodus addon to Kodi by following these steps:

  1. Open Kodi and click on the gear icon to go to Settings.
  2. Select “File Manager” from the list of options.
  3. Click on “Add Source.”
  4. Enter the following path: https://i-a-c.github.io/ and name it “Exodus” and then click “OK”.
  5. Go back to the main menu and select “Addons” then click on the package icon at the top left.
  6. Click on “Install from zip file” and select “Exodus”.
  7. Select “repository.exodusredux.zip” and then wait for the notification confirming the successful installation.
  8. Click on “Install from repository” then select “Exodus Redux Repo”
  9. Select “Video Add-ons” and then click on “Exodus Redux”.
  10. Finally, click on “Install” and wait for the notification confirming the successful installation of the Exodus Kodi addon.

Once Exodus is installed, you can access all the content it has to offer by going to “Addons” from the Kodi home screen and then selecting “Exodus Redux”.
