Answering the Target

Now, let’s dive into the installation process of Covenant on Kodi. First, you need to have Kodi installed on your device. Once you have Kodi up and running, you’ll need to add the source for Covenant. Under System settings, click on File Manager and then Add Source. A dialogue box will appear where you need to enter the following URL - ‘http://archive.org/download/repository.xvbmc/'. After that, add a name for this source under the ‘Enter a name for this media source’, and then click on OK.

Next, go back to the home screen and select Add-ons and then click on the open box icon located in the top left corner. Select Install from zip file, and then select the source name you entered above. Click on repository.xvbmc-4.2.0.zip and let it install. Once installed, a notification will appear on the top right corner. After that, click on Install from repository and select XvBMC (Add-ons) REPOsitory. Select Video add-ons, then Covenant and click on Install. Wait for the add-on to install and once done, a notification will appear on the top right corner.
